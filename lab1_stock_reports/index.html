    Exercise 1 - Existing Stock Report Availability in Splunk

    Objective

    Compare CUIC reports with equivalent Splunk dashboards to demonstrate real-time visibility and data fidelity.

    Overview

    This exercise demonstrates how Splunk is showing real time data and the existing reports from CCE are already available:

    • Real-time data updates
    • Interactive visualizations
    • Drill-down capabilities

    Part A: Agent Real-time Report Comparison

    Step 1: Access CUIC Agent Real-time Report

    1. Login to CUIC
    2. Open CUIC web interface:
    3. After successful login, you will see the CUIC main interface:
    4. Navigate to Reports on left navigation panel
    5. Search for Agent Real Time report in the search bar.
    6. Click on the Agent Real Time report
    7. In the Filter Modal select all agents and click >> icon
    8. Click on Run Button
    9. The report will execute and display the agent real-time data:

    Step 2: Access Equivalent Splunk Report

    1. Login to Splunk
    2. Open Splunk web interface:

    3. After successful login, you will see the Splunk main interface with available apps:

    4. Navigate to CCE Reporting app

    5. You will see the Supervisor Dashboard with key metrics:

    6. Run Agent Real Time Report

    7. Navigate to Dashboards tab
    8. Under Dashboard tab search for "Agent Real Time"
    9. Click on the "Agent Real Time" report link
    10. Splunk will open the Agent Real Time dashboard showing live agent data:

    Step 3: Compare Results

    Compare Results in CUIC and Splunk - Validate if Splunk refreshes the report every 30 seconds and shows fresh data - Validate the data matches with CUIC Reports

    Part B: Skill Group Status Graphical Real-time

    Step 1: Access CUIC Skill Group Status Report

    1. Navigate to CUIC Report
    2. Open CUIC web interface.
    3. Navigate to Reports on left navigation panel
    4. Search for Skill Group Status Graphical Real Time report in the search bar.
    5. Click on the Skill Group Status Graphical Real Time report
    6. In the Filter Modal select All Skill Groups and click >> icon
    7. Click on Run Button

    Step 2: Access Equivalent Splunk Report

    1. Open Splunk web interface

    2. Navigate to CCE Reporting app

    3. Navigate to Dashboards tab

    4. Under Dashboard tab search for "Skill Group Status Graphical Real Time"

    5. Click on the "Skill Group Status Graphical Real Time" report link

    6. Select same filter as CUIC Filter and the folowing report should show

    Step 3: Compare Results

    Compare Results in CUIC and Splunk - Validate if Splunk refreshes the report every 30 seconds and shows fresh data - Validate the data matches with CUIC Reports and graphs are similar

    Part C: Agent Team Attendance Historical

    Step 1: Access CUIC Agent Team Attendance Historical Report

    1. Navigate to CUIC Report
    2. Open CUIC web interface.
    3. Navigate to Reports on left navigation panel
    4. Search for Agent Team Attendance Historical report in the search bar.
    5. Click on the Agent Team Attendance Historical report
    6. In the Filter Modal select custom date range.
    7. Select a start date which is 7 days ago.
    8. Select all Agent Teams by clicking on >> button.
    9. Click on Run Button

    Step 2: Access Equivalent Splunk Report

    1. Login to Splunk
    2. Open Splunk web interface
    3. Navigate to CCE Reporting app
    4. Navigate to Dashboards tab
    5. Under Dashboard tab search for "Agent Team Attendance Historical"
    6. Click on the "Agent Team Attendance Historical" report link
    7. Select same date range and team filters as CUIC Filter (by default last 7 days is selected) and the following report should show
    8. Click on the 1st row to drill-down

    Step 3: Compare Results and Drill-down Capabilities

    Compare Results in CUIC and Splunk: - Validate the attendance data matches between CUIC and Splunk reports - Test drill-down functionality in Splunk to get detailed agent information - Validate historical data accuracy for the selected date range

    Exercise 1 Summary

    Key Comparisons Completed

    1. Agent Real-time Report: Real-time agent status and state information
    2. Skill Group Status Graphical Real-time: Queue and service level monitoring
    3. Agent Team Attendance Historical: Historical attendance and productivity data

    Benefits Demonstrated

    • Data Fidelity: Splunk shows the same accurate data as CUIC reports
    • Real-time Updates: Splunk provides automatic refresh capabilities
    • Enhanced Visualization: More interactive and drill-down options
